Installation Guide

This guide provides detailed instructions for installing Rhodonite on various platforms.

System Requirements

Before installing Rhodonite, ensure your system meets the following requirements:

  • Operating System: Windows 10+, macOS 10.15+, or Linux
  • Node.js: v14.0 or later
  • Memory: At least 4GB RAM
  • GPU: WebGL 2.0 compatible graphics card (recommended)

Installation Methods

Method 1: Using NPM

The easiest way to install Rhodonite is via npm:

npm install @rhodonite/core --save

Method 2: Using CDN

You can also include Rhodonite directly from a CDN:

<script src="https://cdn.example.com/rhodonite/latest/rhodonite.min.js"></script>

Method 3: From Source

For the latest features or to contribute, you can build from source:

# Clone the repository
git clone https://github.com/actnwit/rhodonite.git

# Navigate to the directory
cd rhodonite

# Install dependencies
npm install

# Build the project
npm run build

Verifying Installation

To verify that Rhodonite is installed correctly, create a simple test file:

import * as Rhodonite from '@rhodonite/core';

// Initialize Rhodonite
const system = Rhodonite.System.getInstance();
if (system) {
  console.log('Rhodonite installed correctly!');
}

Note: If you encounter any issues during installation, please check our troubleshooting guide or open an issue on GitHub.

    Class ConstantScalarVariableShaderNode<T>

    A shader node that represents a constant scalar variable. This node extends the base ConstantVariableShaderNode specifically for scalar values, providing functionality to set and manage scalar constant values in shader programs.

    Type Parameters

    • T extends ComponentTypeEnum

      The component type enum that defines the data type of the scalar value

    Class Joint

    generate

    • generate(desc: IAnyPrimitiveDescriptor): void

      Generates a 3D joint visualization as an arrow-like shape connecting two points. The joint is rendered as a combination of pyramidal shapes and connecting lines, creating a visual representation of the connection between this joint and its parent.

      Parameters

      • desc: IAnyPrimitiveDescriptor

        The primitive descriptor containing material and other rendering properties

      Returns void

      Remarks

      The generated shape consists of:

      • A long pyramid extending from this joint towards the parent
      • A connecting plane section
      • A short pyramid at the parent joint position All rendered as line segments for wireframe visualization
